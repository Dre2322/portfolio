Moffat Bay Lodge Website

A full-stack reservation system with authentication, responsive design, and database integration.

Project Overview

This team project delivers a complete website for the fictional Moffat Bay Lodge. Visitors can explore attractions, register and log in, make reservations, and look up past bookings. All reservations and user accounts are stored securely in a MySQL database with Java-based backend services running on Tomcat.

Key Features

  • Landing Page: A marketing-focused entry point to welcome users.
  • About & Contact Pages: Clean, combined content for lodge info and contact details.
  • Attractions Page: A visually engaging carousel of activities and highlights.
  • Authentication: Secure registration & login with validation and password hashing.
  • Reservation System: Choose room type, dates, guests, and confirm bookings.
  • Reservation Lookup: Quickly search past reservations by ID or email.
